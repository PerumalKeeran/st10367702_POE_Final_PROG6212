# st10367702_POE_Final_PROG6212
The Contract Monthly Claim System (CMCS) is a web application developed with ASP.NET Core MVC, Entity Framework Core, and SQL Server. The system aims to simplify the monthly claim process for Independent Contractor lecturers through automated calculations, organization of supporting documents, and a defined approval workflow. Instructors may submit claims along with documentation, Coordinators and Managers carry out distinct approval phases, and HR can view the finally approved claims and utilize CSV export features. The application guarantees precision, clarity, and effectiveness throughout the entire claim-handling process
